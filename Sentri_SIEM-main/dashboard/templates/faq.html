Frequently Asked Questions

1. How do I set up SentriSIEM?

To set up SentriSIEM, ensure that you have all dependencies installed and run the `app.py` script to start the application.

2. Where can I view collected logs?

Collected logs can be viewed under the "Reports" section. You can also query the database directly.

3. How can I filter anomalies?

Go to the "Reports" page and use the interactive filters to display specific anomalies based on time range and severity.

4. How do I export logs?

You can export logs in CSV format from the "Reports" page. Click on the "Export as CSV" button to download the data.

5. How can I secure the system?

Ensure that only admins have access to critical settings by using Role-Based Access Control (RBAC). Also, configure HTTPS for secure communication.

6. How do I reset my password?

Currently, there is no automated password reset feature. Contact the system administrator to update your credentials.
